## Recording Notes

### Screen Recording Setup

1. **Browser**: Chrome with dark mode
2. **Extension**: TIPZ extension installed
3. **Resolution**: 1920x1080 or 2560x1440
4. **Mouse**: Use custom amber cursor for visibility

### Post-Processing

1. Add text overlays in terminal style (JetBrains Mono)
2. Add subtle grid background where appropriate
3. Color grade to terminal aesthetic (boost blacks, amber tint)
4. Add transition effects (glitch, static, fade)

### Audio Options

**Option A: No Audio**
- Works for autoplay on websites
- Twitter muted autoplay compatible

**Option B: Subtle Music**
- Lo-fi electronic/synthwave
- No lyrics
- Fade in at Scene 3, fade out at Scene 9

**Option C: Sound Design**
- Keyboard clicks for typing
- Subtle UI sounds for interactions
- Confirmation chime for success
